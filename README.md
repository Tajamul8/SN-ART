# SN ART – Kashmir Artisan Crafts

A premium heritage catalog website showcasing authentic Kashmiri craftsmanship.

## Features

- 🎨 **Glassmorphism Design** – Modern frosted glass UI effects
- 🎬 **Hero Video Background** – Full-screen orchard video
- 🛍️ **Product Catalog** – Filterable product grid with categories
- 🔍 **Search & Quick View** – Instant search and product modals
- 🛒 **Cart & Wishlist** – Full shopping cart and wishlist functionality
- ✨ **3D Tilt Cards** – Interactive card hover effects
- 📱 **Fully Responsive** – Mobile-optimized with touch support
- 🎭 **Scroll Animations** – Fade-up, slide, and parallax reveals
- 💬 **WhatsApp Integration** – Floating commission inquiry button
- 📰 **Newsletter Signup** – Heritage Chronicle subscription
- 🔢 **Animated Counters** – Dynamic statistics display
- 🎠 **Testimonial Slider** – Auto-rotating patron reviews
- 🖱️ **Mouse Follow Glow** – Interactive cursor effect
- ⏳ **Luxury Page Loader** – Branded loading animation

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Cinzel, Cormorant Garamond, Jost)
- lite-server for local development

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3010](http://localhost:3010) in your browser.

## Collections

- **Kashmiri Tokri Baskets** – Handwoven willow basketry
- **Papier Mâché Artifacts** – Hand-painted lacquer art
- **Pashmina & Kani Shawls** – Fine hand-loomed Pashmina wool
- **Walnut Wood Carvings** – Intricately carved walnut woodwork
